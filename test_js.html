Une file d'attente composé de Client se retrouve à un guichet de banque, chaque client a 2 opérations à effectuer au guichet (Ajout d'argent à un compte, Retrait d'argent). Une fois les opérations terminées, c'est au tour du Client suivant de passer au guichet. Ainsi de suite, jusqu'à que la file soit terminée. La file contient 5 clients. Les opérations sont des additions et soustraction d'un montant stocké dans un compte bancaire. La file d'attente sera un tableau d'objet contenant une instance Client. L'exercice s'exécutera sur un navigateur depuis une page HTML vide.
